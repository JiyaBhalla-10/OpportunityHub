# OpportunityHub

## Discover Tech Opportunities. Build Your Future.

OpportunityHub is a modern, dark-themed web platform designed to help students discover and explore **tech programs, internships, hackathons, scholarships, open-source opportunities, and career resources**—all in one place.

Built with a futuristic UI and student-first mindset, OpportunityHub empowers learners to make informed career decisions and stay ahead in the tech ecosystem.

---

## Live Project
https://opprtunityhub.netlify.app/

---

## Project Motive

Students often struggle to find reliable, centralized, and up-to-date information about tech opportunities relevant to their academic year and career goals.

OpportunityHub was created to solve this problem by providing:

- A centralized hub for verified opportunities  
- Clear categorization by student year and interest  


The goal is to **bridge the gap between students and real-world tech exposure**.

---

## Features

### Opportunity Discovery
- Tech programs and internships
- Hackathons and competitions
- Scholarships and fellowships
- Open-source contribution opportunities
- Career-focused resources

### Smart Browsing Experience
- Year-based filtering (1st–4th year)
- Category-based filtering
- Search functionality
- Clean, readable opportunity cards

## Tech Stack

- **Vite** – fast build tooling
- **React** – component-based UI
- **TypeScript** – type safety and scalability
- **Tailwind CSS** – utility-first styling

## Getting Started (Local Setup)

### Prerequisites
- Node.js  
- npm  
(Recommended: install Node using `nvm`)

---

### Installation
#### Clone the repository
git clone https://github.com/JiyaBhalla-10/OpportunityHub.git

#### Navigate to the project directory
cd OpportunityHub

#### Install dependencies
npm install

#### Start the development server
npm run dev

## Development Workflow

- Develop locally using your preferred IDE
